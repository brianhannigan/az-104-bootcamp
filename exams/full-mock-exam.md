## Questions

### 1
A user needs to manage resources in a resource group but must not assign permissions. Which role should you use?

A. Owner  
B. Contributor  
C. Reader  
D. User Access Administrator  

**Answer:** B

---

### 2
You need to require a tag on every resource in a resource group. What should you use?

A. RBAC  
B. Lock  
C. Azure Policy  
D. NSG  

**Answer:** C

---

### 3
You need to connect two VNets privately in Azure. What should you use?

A. NSG  
B. VNet Peering  
C. Route Table only  
D. Public IP  

**Answer:** B

### 4

A. DNS issue  
B. Lock
