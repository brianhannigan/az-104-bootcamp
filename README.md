# AZ-104 Bootcamp (10-Day Intensive)

This repository is a focused, exam-driven AZ-104 training system designed for rapid preparation and practical Azure skill-building.

## Goals

- Pass AZ-104 fast
- Focus only on what is tested
- Build real Azure administration muscle memory
- Prepare for transition into AZ-500

## Repository Structure

| Folder | Purpose |
|---|---|
| `days/` | Daily training plans |
| `assets/` | Diagrams and mental models |
| `exams/` | Cheat sheets, gotchas, and mock exam |
| `scripts/` | Helper scripts for repo setup or labs |

## 10-Day Study Flow

| Day | Focus |
|---|---|
| Day 1 | Identity, RBAC, Governance |
| Day 2 | Networking Core |
| Day 3 | Advanced Networking |
| Day 4 | Storage |
| Day 5 | Compute |
| Day 6 | App Services and Containers |
| Day 7 | Monitoring and Backup |
| Day 8 | Scenario Practice |
| Day 9 | Weak Area Reinforcement |
| Day 10 | Final Review and Mock Exam |
