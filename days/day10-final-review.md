# 🔹 STEP 1 – MASTER CHEAT SHEET (CRITICAL)

## 🔐 Identity & Governance

- RBAC = WHO  
- Policy = WHAT  
- Lock = PROTECTION  

---

## 🌐 Networking

- NSG = allow/deny traffic  
- UDR = control routing  
- Peering = NOT transitive  
- Private Endpoint = private IP  

---

## 📦 Storage

- Blob = object  
- File = shared  
- Disk = VM  

- RBAC > SAS > Keys  

- LRS = local  
- ZRS = zones  
- GRS = region  

- Hot / Cool / Archive  

---

## 🖥️ Compute

- VM = full control  
- App Service = managed  

- Availability Set = same DC  
- Availability Zone = different DC  

- VMSS = scaling  

---

## 📊 Monitoring

- Logs = deep analysis  
- Log Analytics = required  

- Alert → Action Group
